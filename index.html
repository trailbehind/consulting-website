Some of our clients need an invention,
and some need an invention built.

1.Waterfall

For projects with detailed specifications, we build, test, and release using more of a waterfall approach, but with more iterations and earlier previews of software in development.

2. Iteration

For greenfield projects, particularly to help start-ups in the pre-launch phase, we follow more of an agile methodology, with quick mock-ups and working prototypes built at the earliest moments, and built and rebuilt until user engagement metrics are off the charts.

Creations

App: Gaia GPS

Recently featured by Apple, Gaia GPS is the ultimate app for the backcountry. Across iOS, Android, and the web, Gaia GPS provides a complete platform for mapping and exploration, and showcases our skill in building powerful applications.

Service: Cloud Sync API

Built on Postgres, Django, and Amazon EC2, our cloud-sync API enables many of the apps we built to synchronize data between mobile devices and the web. Thousands of users per day sync data to the Cloud services we built.

Website: UnchartedApps.com

Built on Jekyll and GitHub pages, with a dash of Jquery, we hope this website demonstrates a modern design sensibility and an elegant solution to the problem of letting you know what we do.
