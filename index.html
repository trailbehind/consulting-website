To cut a new path, inquire within.

Get in touch »

We can execute a big spec or spec a big idea, and we know when to do which.

For projects with detailed specifications, pixel-perfect designs, and firm time constraints, we build, test, and release with a more classic ideas, based on milestones and deliverables. At the same time, we show the work in progress, and provide clients with the greatest possible leeway for change and innovation.

For clients seeking UX design and innovation, we focus the methodology more on quick mock-ups and working prototypes, built at the earliest moments, and rebuilt iteratively until user engagement metrics are off the charts.

Creations

App: Gaia GPS

Recently featured by Apple, Gaia GPS is the ultimate app for the backcountry. Across iOS, Android, and the web, Gaia GPS provides a complete platform for mapping and exploration, and showcases our skill in building powerful applications.

Service: Cloud Sync API

Built on Postgres, Django, and Amazon EC2, our cloud-sync API enables many of the apps we built to synchronize data between mobile devices and the web. Thousands of users per day sync data to the Cloud services we built.

Website: UnchartedApps.com

Built on Jekyll-Bootstrap and GitHub pages, with a dash of Jquery, we hope this website demonstrates a modern design sensibility and an elegant solution to the problem of letting you know what we do.

Web and Mobile

We build apps on one or sometimes across all of iOS, Android, and the web. We can effectively build native mobile apps, that also work responsively on the web.

Mapping Libraries

We have a unique depth of experience working with many of the best mobile mapping toolkits of the day, including Maply/WhirlyGlobe, Route-me, Apple Mapkit, Google Maps, and Nutiteq. On the web, we have deployed Leaflet, OpenLayers, and Google Maps.

Web Design and Stack

We use the best technologies for creating beautiful and functional websites, including HTML5, Javascript, Angular.js, and Bootstrap. On the back-end, we tend to use Amazon EC2, Django/Python, and Postgres, but have experience in many others.
